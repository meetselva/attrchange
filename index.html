Overview

attrchange is a simple jQuery function to bind a listener function to any HTML element on attribute change.

This plugin can be used to monitor the changes happening to a HTML element by some other code.

The plugin was originally written to answer a question posted on stack overflow

Basic Usage

Start using this plugin by following 2 simple steps.
  • Include attrchange.js to your web page 
    <script type="text/javascript"
        src="https://raw.github.com/meetselva/attrchange/master/attrchange.js"></script>
  • Bind attrchange handler function to selected elements 
    $(selector).attrchange(function () {
    //Triggered when the selected elements attribute is added/updated/removed
});

Example:

Example 1: A simple example to demonstrate the execution of the onattrchange handler function.

In the demo, each block's top position is increased/decreased by 5px every second which triggers the attrchange handler. Inside the handler is a simple logger which displays the name of the attribute along with the block number.

View code
HTML: 
<div id="blocks">
     <div title="b1" class="block" style="top: 65px;">b1</div>
     <div title="b2" class="block" style="top: 65px;">b2</div>  
</div>
JS: 
$('#blocks .block').attrchange(function(e, attrName) {
  $('#example1 .logger').prepend('<p>- attribute <b>' + attrName + '</b> changed for <b>' + 
  		        this.title + '</b></p>');
});

Example 2: An interactive demo to show the execution of the onattrchange handler function.

The interactive demo allows you to add/update/remove attributes on the demo div elements. Click on the demo div to pop the Attribute Changer dialog, using which you can add/update the attributes of the selected div.

Upon each update the handler function will be triggered and In this example the handler function simply prints the attribute name and the div id in the logger.
